# Thiago Gabry's Resume (ML Portfolio)

Single-page React resume site highlighting real production ML outcomes (projects, experience, skills, and contact) in a polished, recruiter-friendly format.

## Resume-ready highlights
- Built an animated resume/portfolio UI with modular sections: `Hero`, `Projects`, `Skills`, `How I Work`, `Experience`, and `Contact`.
- Communicate impact with measurable outcomes (e.g., `80ms -> 45ms p99`, `10K+ req/s`, `2M+ images/day`, `3x faster iterations`).
- Delivered a responsive UX: fixed `Navbar` with mobile menu and smooth motion via `Framer Motion`.
- Designed for maintainability using reusable `shadcn/ui`-style components (Radix + Tailwind) and a consistent layout system.

## Tech stack
- `Vite` + `React` + `TypeScript`
- `Tailwind CSS` (with `shadcn/ui`-style components / Radix primitives)
- `Framer Motion`, `React Router`, `Lucide` icons
- `@tanstack/react-query` (app-level provider), `Sonner` toast UI

## Local development
- `npm install`
- `npm run dev`
